Getting Started

The CosmoZoom API provides programmatic access to NASA space images, AI pattern recognition, and advanced analysis tools.

// Base URL
https://api.cosmozoom.space/v1

// Example Request
fetch('https://api.cosmozoom.space/v1/images/jwst/latest')
  .then(response => response.json())
  .then(data => console.log(data));

Authentication

All API requests require an API key. Include your key in the Authorization header:

headers: {
  'Authorization': 'Bearer YOUR_API_KEY',
  'Content-Type': 'application/json'
}

Image API

GET /images

Retrieve space images from various sources

GET
/images

Query Parameters:

  • source - Data source (jwst, hubble, usgs)
  • type - Object type (galaxy, nebula, star, planet)
  • limit - Number of results (default: 10)
// Example: Get JWST galaxy images
fetch('https://api.cosmozoom.space/v1/images?source=jwst&type=galaxy&limit=5')
  .then(response => response.json())
  .then(data => console.log(data));

AI Analysis API

POST /analyze

Run AI pattern recognition on an image

POST
/analyze
// Example: Analyze an image
fetch('https://api.cosmozoom.space/v1/analyze', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    imageUrl: 'https://example.com/space-image.jpg',
    analysisType: 'full'
  })
})
.then(response => response.json())
.then(data => console.log(data.detections));

Code Examples

JavaScript Example

// Complete example: Load and analyze an image
async function analyzeSpaceImage() {
  const apiKey = 'YOUR_API_KEY';
  
  // Get latest JWST image
  const imageResponse = await fetch(
    'https://api.cosmozoom.space/v1/images/jwst/latest',
    { headers: { 'Authorization': `Bearer ${apiKey}` } }
  );
  const imageData = await imageResponse.json();
  
  // Run AI analysis
  const analysisResponse = await fetch(
    'https://api.cosmozoom.space/v1/analyze',
    {
      method: 'POST',
      headers: {
        'Authorization': `Bearer ${apiKey}`,
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        imageUrl: imageData.url,
        analysisType: 'full'
      })
    }
  );
  
  const analysis = await analysisResponse.json();
  console.log('Detected objects:', analysis.detections);
}

analyzeSpaceImage();
